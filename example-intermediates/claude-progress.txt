ARCHITECTURE OVERVIEW
================================================================================

Frontend: http://localhost:5173 (Vite dev server)
Backend:  http://localhost:3001 (Express)
Database: server/data/claude_clone.db (SQLite, auto-created)
API Key:  /tmp/api-key (server-side only, never sent to browser)

Three-column layout:
  [Sidebar 260px] | [Chat area flex-1] | [Artifact panel 480px, conditional]

Streaming: SSE via fetch() async generator (streamChatGenerator in api.js)
  - POST /api/claude/chat/stream returns text/event-stream
  - Events: message_start, text (delta), done (final), error

Artifact detection: <antArtifact identifier="..." type="..." title="...">
  - Types: code, html, svg, react, mermaid, text
  - Parsed on client by ChatArea after stream completes

Auth: Single default user "default-user" (no real auth needed)
  - requireAuth middleware reads X-User-Id header or defaults to "default-user"
